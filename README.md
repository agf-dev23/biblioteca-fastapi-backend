# 📚 Biblioteca API - FastAPI Backend

REST API for managing a personal library system.

## 🚀 Tech Stack

- FastAPI
- MySQL
- SQLAlchemy
- Alembic
- Python 3.11+

## 📦 Features

- CRUD de libros
- Arquitectura por capas (API, Services, Models)
- Migraciones con Alembic
- Validación con Pydantic

## ⚙️ Instalación

```bash
git clone https://github.com/agf-dev23/biblioteca-fastapi-backend.git
cd biblioteca-fastapi-backend

python -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
